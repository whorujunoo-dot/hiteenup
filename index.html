모든 기록을 파일로 내보내거나, 백업 파일에서 복원할 수 있어요.

📖 사용 매뉴얼 ▼

1. 처음 시작하기

  • 처음 앱을 켜면 아이 이름, 생년월일, 성별, 몸무게를 입력해요.
  • 나이는 자동으로 만 나이로 계산됩니다.

2. 매일 입력해요 (입력 탭)

  • 잠든 시각, 일어난 시각, 앱 사용 시간, 운동 시간을 입력해요.
  • 오늘의 메모는 자유롭게 적어두면 나중에 다시 볼 수 있어요.

3. 키·몸무게는 원할 때마다

  • 매일이 아니어도 됩니다. 측정한 날 입력하면 그래프가 자동으로 그려져요.

4. 또래와 비교 (홈 탭)

  • 한국 표준성장도표(질병관리청 기준)와 비교해서 +/- cm로 보여드려요.
  • 키를 입력하면 자동으로 갱신됩니다.

5. 키 크는 정보 (정보 탭)

  • 연령대별·주제별로 정보를 필터링할 수 있어요.
  • 제목을 누르면 본문이 펼쳐집니다.

6. 백업

  • 설정 → 백업/복원에서 JSON 파일로 내보낼 수 있어요.
  • 휴대폰을 바꾸거나 앱을 다시 설치한 경우 이 파일로 복원하세요.

7. 날짜 자동

  • 모든 기록의 날짜는 휴대폰 시각 기준으로 자동 저장됩니다. 수정할 수 없어요.
ℹ️ 데이터 안내 ▼

또래 비교 기준은 질병관리청 「2017 한국 소아청소년 성장도표」의 50백분위수(평균)를 기준으로 합니다.
